# 🐍 Snake Game (Single HTML File)

A simple and visually appealing Snake Game built using pure HTML, CSS, and JavaScript — all in one file. The game runs directly in the browser with no dependencies.

## 🎮 Features

* Smooth and responsive snake movement
* Score tracking system
* Clean and modern UI design
* Fully playable in any web browser
* Lightweight (single HTML file)

## 🕹️ How to Play

* Use **Arrow Keys** to control the snake
* Eat food to increase your score
* Avoid hitting the walls or yourself

## 🚀 Getting Started

1. Download or clone this repository
2. Open the HTML file in your browser
3. Start playing instantly!
